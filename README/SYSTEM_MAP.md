# HG Film Producer — systemkart

Dette dokumentet beskriver hovedsystemene i HG Film Producer.

## 1. Studio

Studioet er spillerens base.

Systemet bør dekke:

- penger
- rykte
- ansatte
- kontorer
- utstyr
- produksjonskapasitet
- avtaler
- distribusjon
- langsiktig strategi

## 2. Filmprosjekt

Hver film er et prosjekt med egen livssyklus.

```text
idé → manus → preproduksjon → innspilling → klipp → lansering → etterliv
```

Filmprosjektet samler alle valg spilleren tar.

## 3. Manus

Manus-systemet skal gjøre skriving spillbart.

Elementer:

- idé
- sjanger
- tema
- hovedperson
- mål
- konflikt
- antagonist
- vendepunkter
- scener
- dialog
- slutt
- omskriving

## 4. Crew

Crew-systemet handler om rollene bak filmen.

Roller:

- produsent
- regissør
- manusforfatter
- fotograf
- klipper
- komponist
- lyddesigner
- scenograf
- kostyme
- casting
- location manager
- markedsføring

Hver rolle skal påvirke filmen mekanisk.

## 5. Skuespillere

Skuespillere skal ha egenskaper som påvirker filmens uttrykk.

Mulige egenskaper:

- stjerneverdi
- erfaring
- spillestil
- kjemi
- sjangerstyrke
- improvisasjon
- disiplin
- publikumstrekk

## 6. Locations

Locations er koblingen til History Go.

Et sted kan påvirke:

- stemning
- sjanger
- produksjonskostnad
- autentisitet
- logistikk
- visuell verdi
- historisk verdi
- publikumsgjenkjenning

## 7. Produksjon

Produksjonssystemet dekker selve innspillingen.

Mulige hendelser:

- dårlig vær
- forsinkelser
- budsjettpress
- tekniske problemer
- kreativ konflikt
- god improvisasjon
- location-problemer
- uventet sterk scene

## 8. Klipp og postproduksjon

Postproduksjon handler om å gjøre råmateriale til film.

Elementer:

- rytme
- struktur
- lyd
- musikk
- farge
- effekter
- trailer
- testvisning
- endelig versjon

## 9. Lansering

Lansering avgjør hvordan filmen møter verden.

Kanaler:

- festival
- kino
- streaming
- TV
- skole/kultur
- internasjonalt salg

## 10. Publikum og kritikk

Filmen får respons fra ulike grupper.

Mulige responsfelt:

- publikumstall
- kritikerrespons
- kultstatus
- festivalverdi
- økonomisk resultat
- studio-rykte
- sjangerpublikum

## 11. Mentorer

Mentorer er historiske og faglige rådgivere.

De hjelper spilleren å forstå roller, teknikker og valg.

Mentor-systemet skal koble:

```text
rolle → problem → eksempel → teknikk → spillvalg
```

## 12. Filmhistorie

Filmhistorie skal brukes som et aktivt system.

Eksempler på kunnskapsfelt:

- epoker
- sjangre
- bevegelser
- teknikker
- roller
- filmer
- personer
- steder
- produksjonsmodeller

## 13. Filmleksikon

Leksikonet er ikke separat fra spillet.

Det skal forklare ting spilleren faktisk bruker.

Eksempel:

```text
Spilleren bruker jump cut → leksikon forklarer jump cut → teknikken blir del av studioets språk.
```

## Hovedregel

Alle systemer skal støtte samme kjerne:

```text
Lag bedre film ved å forstå film bedre.
```
