# 1950s Asian postwar verification

This batch continues source-backed quality control of existing Production Cases. It adds no films to the catalogue.

## Verified films

- Rashomon (1950)
- Tokyo Story (1953)
- Seven Samurai (1954)
- Pather Panchali (1955)

## Production systems compared

- Daiei period modernism using contradictory testimony, repeated action, forest light and rain;
- Shochiku domestic drama using low fixed framing, restrained performance and transitional everyday space;
- Toho historical action using ensemble preparation, researched village geography, weather and multiple cameras;
- Bengali location humanism using limited resources, seasonal observation, family routine and non-star performance.

Each film has a source-backed history profile, a complete 17-area audit, three playable historical alternatives and four inspectable institutional or archival sources.

The unified verification registry reaches 72 source-backed Production Cases after this batch. Catalogue size remains dynamic and unchanged by this work.
