# Priority independent-film completion

## Why this correction exists

The user explicitly prioritized a 27-film American independent-cinema list before the later Asian, European festival and documentary expansion.

The first modern expansion added seven of those films, but it also substituted twenty different titles. Those twenty substitute titles remain valid additions to Filmverket. This completion does not remove them.

Instead, this batch adds the twenty originally prioritized films that were still absent, so the requested 27-film list is now complete exactly once.
