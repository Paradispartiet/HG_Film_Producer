# Production Case forecast acceptance

The pre-choice production forecast is accepted when:

1. every production choice shows the projected budget, time and creative-control position after that choice;
2. projections count only earlier phases plus the current candidate choice;
3. switching a choice replaces the previous impact instead of adding it twice;
4. low-margin, budget-overrun, schedule-overrun and overextended states are clearly identified before selection;
5. overrun copy states the possible Producer or Assistant result cap;
6. risky choices remain selectable;
7. the existing choice-quality score and saved progress format remain unchanged.
