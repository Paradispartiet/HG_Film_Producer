# 1940s noir and realism verification

This batch continues source-backed quality control of existing Production Cases. It adds no films to the catalogue.

## Verified films

- Double Indemnity (1944)
- Brief Encounter (1945)
- Rome, Open City (1945)
- The Third Man (1949)

## Production systems compared

- Paramount studio noir using confession, procedure, chiaroscuro and fatal performance;
- British romantic melodrama using repeated railway space, interior narration and musical memory;
- Italian neorealism emerging from occupation experience, street production and material scarcity;
- British postwar location noir using occupied Vienna, multiple units, reconstructed sewers and solo zither.

Each film has a source-backed history profile, a complete 17-area audit, three playable historical alternatives and four inspectable institutional or archival sources.

The unified verification registry reaches 68 source-backed Production Cases after this batch. Catalogue size remains dynamic and unchanged by this work.
